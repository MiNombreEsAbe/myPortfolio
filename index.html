A* Pathfinder Visualizer with Javascript

Project Description: This project visualizes the A* pathfinding algorithm. It will always find the shortest path from the starting postition (green square) to the ending position (orange square) while taking into account any barriers (black squares).

How To Use: On the top right, there are 4 buttons ("Toggle Barriers", "Set Start", "Set End", "AStar", "Reset"). Click any of the first three to set barriers, starting position, and ending position. After that, click on "AStar" to find the path. At any point, you can click on "Reset" to make a clean slate.
